見出し2下の文章

## 改行

段落は一行空ける

改行は空白を２つ重ねる　　
重ねないと
連続した文章と見なされ一行に表示される  

強調したい時は *アスタリスク* か _アンダーバー_ で囲う  
**2つ** にすると __さらに__ 強くなる  

インラインコードは `use back_quotation` でできる  
複数行になるなら  
    blank_of_head = 4
    行頭に４空白
    comment //コメント


リスト表示　空白を忘れない
* リスト1
* リスト2
* リスト3

1. 番号リスト1
2. 番号リスト2
3. 番号リスト3
